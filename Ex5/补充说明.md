Exercise 5 环境配置补充说明 
一、Nodejs 安装 
直接官网下载安装即可。  
二、Ganache CLI 安装 1.  在命令行中运行npm install -g ganache-cli 来安装Ganache CLI 2.  在命令行中运行ganache-cli 来运行节点（使用Ctrl-C 来停止节点）  
三、remix 使用 1.  在网页浏览器中打开https://remix.ethereum.org 2.  Workspaces 选择playground          3.  Environment 选择Custom - External Http Provider      
此步骤只有在成功安装并运行ganache-cli 的情况下才可正常运行。后续External 
HTTP Provider Endpoint 输入http://127.0.0.1:8545（默认值）  
四、编译运行Solidity 代码 
此处假设sol 代码文件名为BlockchainSplitwise.sol。 1.  编译Solidity 代码 
         2.  复制ABI 至代码指定位置          3.  对合约进行部署 
         4.  复制合约地址至代码指定位置           
五、其他环境配置相关 
网页左下角“设置”处可以调整界面颜色主题、语言，同学们按需调整就好。
